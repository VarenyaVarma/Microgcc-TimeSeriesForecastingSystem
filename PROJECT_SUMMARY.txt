HOW TO GET STARTED
==================================================

STEP 1: Install Dependencies
$ pip install -r requirements.txt
(Takes 5-10 minutes depending on internet speed)

STEP 2: Generate Data & Train Models
$ python run_pipeline.py
(Takes 10-20 minutes depending on hardware)

This will:
✓ Generate synthetic sales data
✓ Preprocess and clean data
✓ Engineer features
✓ Split train/test temporally
✓ Train all 4 models
✓ Evaluate and select best model
✓ Generate 8-week forecasts
✓ Create visualizations
✓ Save all outputs

STEP 3: Start API Server
$ python -m uvicorn api.app:app --reload
OR
$ uvicorn api.app:app --port 8000

Then visit: http://localhost:8000/docs

STEP 4: Explore Results
- Check outputs/forecasts/ for CSV predictions
- View outputs/plots/ for visualizations
- Review outputs/metrics/ for evaluation metrics

==================================================
EVALUATION METRICS
==================================================

MAE (Mean Absolute Error)
- Average absolute deviation
- Expected error: ~250-350 units
- Interpretable business metric

RMSE (Root Mean Squared Error)
- Primary selection metric
- Penalizes large errors
- Expected: ~300-400 units

MAPE (Mean Absolute Percentage Error)
- Scale-independent error
- Expected: 5-8%
- Business-friendly (%)
